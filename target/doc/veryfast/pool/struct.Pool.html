fn new(align_to_cache: bool, batch: usize) -> Pool<TYPE>

Creates a new Pool.

  • align_to_cache: Should each object be on a separate CPU cache line. Speeds up multithreaded usage but requires a bit more memory in most cases.

  • batch: How many objects should be allocated each time. Higher numbers are faster, but can cause wasted memory if too little are actually used.
